ATOM, A MELHOR IDE QUE RODA NO SEU BROWSER

Atom é um editor de texto de código aberto disponível para as plataformas Linux, macOS e Microsoft Windows, desenvolvido pelo GitHub sob a licença MIT. A ferramenta foi criada usando Electron e tecnologias web como HTML, JavaScript e CSS. Permite a instalação de extensões desenvolvidos com Node.js.
